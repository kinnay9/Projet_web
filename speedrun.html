Le speed run c’est quoi ? :

Le speed run est la pratique de finir un jeu le plus rapidement possible en utilisant tous les manières possibles, y compris le bug ou glitch (un bug ou un glitch est une erreur de programmation dans le jeu)

Cette pratique n’est pas très développée en France, mais il existe quand même une petite communauté. La plus grosse communauté est les anglophones.

Ces communautés restreintes permettent de partager beaucoup d’information entre eux et de connaitre tout le monde de loin ou de près et cela permettent une très bonne entente entre les speed runners.

Pour plus de diversité les speed runners ont créé plusieurs catégories de speed run :

  1. Any% = Ici le but est de finir le plus rapidement possible en utilisant n’importe quel bug (c’est la catégorie la plus populaire)
  2. 100% = Dans cette catégorie ont doit finir le jeu à 100% c’est-à-dire prendre tous les objets cachés et tous les niveaux même optionnels.
  3. Low% = Là l’objectif est d’avoir le moins de pourcentage possible c’est-à-dire récupérer le moins d’objets possible et le moins de niveaux possible.
  4. Multi-segment =Le speed runner peut faire sont speed run en plusieurs partie il peut donc s’arrêter n’importe quand (souvent utilisé dans les très long speed run 10h par exemple).
  5. Single-segment = Simplement finir le jeu d’un seul trait.

Voilà les 5 principales catégories mais il en existe d’autre comme finir le jeux les yeux bander

Ensuite voici quelque terme technique utile dans le speed run :

  1. Damage boost = Se prendre des dégâts pour avancer plus rapidement ou sauter plus haut.
  2. Oob (out of bound) = Aller au-delà de la limite de la carte du jeu c’est-à-dire sortir du jeu pour « prendre un raccourci ».
  3. Save warp = sauvegarder et charger le jeu très vite pour faire passer certaine cinématique.
  4. Zip = passer à travers un mur.

Cette pratique peut paraitre simple au premier abord assez simple mais cela demande une maitrise parfaite du jeu et presque aucune erreur lors de l’exécution du speed run. Pour quelque exemple prenons un jeu assez connu Super Mario Bros en temps normale le jeu ce fini en 4/6 heures environs mais le raccord du monde en speed run est 4m 56s. Ce qui est environs 60 fois moins.
